ประวัติ แรงบันดาลใจ ทักษะ ผลงาน เป้าหมาย

1. ประวัติส่วนตัวและการศึกษา

ชื่อ: นายพิภัช อมาตยกุล

วันเกิด: 25 มิถุนายน พ.ศ. 2552

โรงเรียน: โรงเรียนบุญวาทย์วิทยาลัย

แผนการเรียน: Gifted Computer

ระดับชั้น: มัธยมศึกษาปีที่ 4

เกรดเฉลี่ย: 3.5

วิชาที่ถนัด: คอมพิวเตอร์ คณิตศาสตร์ และวิทยาศาสตร์

ความสนใจ: ด้านคอมพิวเตอร์

2. แรงบันดาลใจและความสนใจด้านไอที

  • เริ่มสนใจด้านไอทีจากการใช้งานคอมพิวเตอร์และเทคโนโลยีในชีวิตประจำวัน
  • มีประสบการณ์แรกในการเขียนโปรแกรมจากการเรียนในแผน Gifted Computer
  • ชอบการเขียนโปรแกรมและการแก้ปัญหาด้วยเหตุผล
  • มองว่าเทคโนโลยีช่วยพัฒนาคุณภาพชีวิตและการศึกษา
  • เลือกศึกษาต่อด้านไอทีเพื่อนำความรู้ไปพัฒนาตนเองและสังคม

3. ทักษะและความสามารถด้านไอที

HTML (ดี)
CSS (ปานกลาง)
JavaScript (ปานกลาง)
PHP (พื้นฐาน)
Python (พื้นฐาน)
  • มีทักษะการคิดเชิงตรรกะและการแก้ปัญหา
  • สามารถเรียนรู้เทคโนโลยีใหม่ด้วยตนเอง

4. ผลงานและกิจกรรมด้านไอที

  • ชื่อผลงาน: เว็บไซต์แนะนำตัว
  • วัตถุประสงค์: เพื่อฝึกการเขียนเว็บไซต์
  • เครื่องมือที่ใช้: HTML, CSS, JavaScript
  • บทบาท: ออกแบบและเขียนโค้ดด้วยตนเอง
  • ปัญหา: การจัดวางหน้าเว็บให้รองรับมือถือ
  • การแก้ไข: ใช้ Responsive Design
  • สิ่งที่ได้เรียนรู้: การออกแบบเว็บให้ใช้งานได้จริง

5. เป้าหมายและเหตุผลในการศึกษาต่อ

  • เลือกมหาวิทยาลัยเชียงใหม่เนื่องจากมีชื่อเสียงด้านเทคโนโลยี
  • หลักสูตรสอดคล้องกับความสนใจด้านไอที
  • ต้องการพัฒนาความรู้ด้านคอมพิวเตอร์อย่างลึกซึ้ง
  • เป้าหมายในอนาคตคือประกอบอาชีพด้านเทคโนโลยีสารสนเทศ
  • มุ่งมั่นพัฒนาตนเองหากได้รับโอกาสเข้าศึกษา
